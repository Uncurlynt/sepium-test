# sepium-test

Тестовое задание на должность Frontend-разработчик / HTML-верстальщик в компанию Сепиум Групп. Работоспособность проверялась в браузерах Safari, Chrome, Firefox на десктопе и телефоне

---

## Выполнено по ТЗ

- [x] **Сетка каталога** (2 колонки на десктопе, 1 колонка на телефоне)
- [x] **Карточка товара**
- [x] **Слайдер изображений** на карточке
  - [x] стрелки навигации (десктоп)
  - [x] пагинация
- [x] **Автопрокрутка слайдера на мобильных** (на десктопе отключена)
- [x] **Кнопка увеличения/просмотра** — открытие галереи изображений
- [x] **Ховер-описание** на карточке (на десктопе)
- [x] **Кнопка “Заказать”** (на телефоне всегда видна, на десктопе появляется при наведении)
- [x] **Избранное**
  - [x] переключение состояния (active/inactive)
  - [x] обновление счётчика
- [x] **Переключение материалов** (один активный вариант, `aria-pressed`)
- [x] **Клик по карточке** ведёт по ссылке из `data-href`, при этом клики по внутренним кнопкам/ссылкам/пагинации не триггерят переход
